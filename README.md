# zktube工具

centos版本

```shell
rm -rf zkt.sh && wget https://download.swarmeth.org/zkt/centos/zkt.sh && chmod a+x zkt.sh && ./zkt.sh
```

根据需求选择：

```
./zkt.sh 
********************************************
(1) 安装docker和docker-compose
(2) 安装单节点zktube
(3) 复制单节点zktube，多开节点
(4) 重启zktube节点
(5) 停止并删除zktube节点
(Q/q) 退出
********************************************
请输入你的选项(1|2|3|4|5|q|Q):
```
